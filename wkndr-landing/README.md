# WKNDR — landing page

Marketing landing page for **WKNDR**, the weather-aware weekend discovery engine for Amsterdam.

A single self-contained `index.html` (no build step, no dependencies). Maneken-inspired layout — warm off-white, floating pill nav, big tight headlines, auto-scrolling pick-cards — built on WKNDR's brand tokens (Helvetica Neue, paper `#faf8f3`, accent `#ff4d1f`).

**Signature touch:** the ambient hero field, weather chip, and CTA gradient cycle live through WKNDR's five weather modes (Hot / Warm / Cool / Cold & wet / Volatile), demonstrating the core idea — *the weather re-ranks everything* — right on the page. Click any mode card to preview it. Respects `prefers-reduced-motion`.

## Run locally

```bash
python3 -m http.server 4184
# → http://localhost:4184
```

Or just open `index.html` in a browser.
